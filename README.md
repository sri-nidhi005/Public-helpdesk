# Public-helpdesk
📞 Public Helpdesk System
📌 Project Overview

The Public Helpdesk System is a web-based application that seeks to enhance the ease of registering, monitoring, and addressing public complaints. The system allows users to file their grievances on a convenient platform while at the same time being able to monitor and address these issues in a much more effective manner.

🎯 Objectives
To act as a central portal for registration of complaints
To improve communication between user and authority figures
To create transparency in the management of issues filed
To lessen administrative workload and reduce response time
🚀 Features
🧑‍💻 User Registration & Login
📝 File Complaints / Grievances
📊 Monitor Issue Status (Open / Under Processing / Resolved)
🔔 Notification alerts
🛠️ Admin dashboard for managing complaints
🔍 Search functionality
